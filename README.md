## Setup Instructions
to find versions for requirements.txt git bash on 
Project_Expense_Tracking and run pip freeze | grep [package_name]. do it once per package

to install all packages with the right version in one shot, 
git bash on Project_Expense_Tracking and run 
pip install -r requirements.txt (or .\requirements.txt)

# Expense Management System
this is an expense management system application, using streamlit for the Frontend
and FastAPI for the Backend

## Project Structure
- **frontend/**: Contains Streamlit app code
- **backend/**: Contains the FastAPI backend server code
- **tests/**: Contains the test cases for both frontend and backend
- **requirements.txt**: Lists the required python packages
- **README.md**: Provides an overview and instructions for the project
